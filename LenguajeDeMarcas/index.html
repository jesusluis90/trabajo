Manual HTML

HTML es un lenguaje para la creacion de paginas web a traves de las etiquetas, las cuales se escriben entre los simbolos de "<" y ">"(menor/mayor que), y con ellas podemos crear entre otras cosas parrafos, insertar tablas, imagenes o enlaces a páginas web. Con este documento se pretende guiar a los lectores para la creacion de un documento simple de HTML.

Estructura basica

Todo documento HTML ha de tener una determinada serie de etiquetas para que este bien formada y funcione correctamente, ademas de que todas las etiquetas han de abrirse y cerrarse, para cerrar basta con escribirla con una contrabarra. Por ejemplo:<h1>Titulo/<h1>

Aqui tendriamos un documento HTML muy basico, con las etiquetas esenciales y todas cerradas adecuadamente: 

			<!doctype html>
			<html>
				<head>
					<title Titulo del documento </title>
				</head>
				<body>
					Cuerpo del documento.
				</body>
  • <html> en esta etiqueta se define que lo utlizado aqui es codigo html.
  • <!doctype Html> con esta etiqueta se indica el tipo del documento, en este caso html.
  • <head> con esta etiqueta indicamos la cabecera, en ella tiene que estar obligatoriamente el titulo del documento con la etiqueta <title>.
  • <body> dentro de esta etiqueta estará el cuerpo del documento, donde introduciremos todo lo demas.

Titulos y secciones

Con la etiqueta hnum (donde num sera un numero del 1 al 6 segun el tamaño que deseemos) modificaremos el texto para indicar que se trata de un titulo.

Las etiquetas article y section nos ayudaran organizar el documento en distintas secciones para mayor legibilidad

Con header crearemos el encabezado en el documento.

Con footer crearemos una zona para el pie de página en nuestro documento.

Elementos de agrupación

Parrafos: Con la etiqueta p podremos definir en nuestro documentos parrafos, teniendo en cuenta que por defecto los saltos de linea y espacios se comprimen.

Textos preformateados: Con la etiqueta pre crearemos textos preformateados, es decir, con un tipo de fuente fijo y respetando saltos de linea y espacios

Listas:Para crear los elementos de una lista tienen que estar con la etiqueta li anidada en cualquiera de estas etiquetas para crear listas:

  • Con ul crearemos listas desordenadas, solo precedidas de un circulo negro, pudiendose modificar esto con los atributos.
  • Con ol crearenos listas ordenadas, por defecto de forma numerica, aunque se pueden cambiar con varios atributos este orden.

Para tenerlo mas claro mostraremos el codigo html para ambas listas:

		Lista desordenada:				Lista ordenada:
		<ul>					<ol>
  			<li> Elemento de lista 1		 	 <li> Elemento de lista 1
  			<li> Elemento de lista 2		  	 <li> Elemento de lista 2
		</ul>					</ol>

Otros tipos de listas que existen son:

Listas de definicion:Con dl crearemos listas de definición, con dt indicaremos el término y con dd su definición.

Linea horizontal: Con hr se crea una linea horizontal para separar diferentes secciones.

Elementos semanticos

Resaltados:

  • i: Bastardilla o Italica
  • b: Negrita
  • u: Subrayado
  • em: Texto con énfasis
  • strong: Texto importante
  • small: Texto pequeño

Salto de linea: Con la etiqueta br crearemos un salto de linea, normalmente usado dentro de los parrafos.

Hipervinculos

Para crear hipervinculos a otras paginas web tienen que crearse con la etiqueta a seguido del atributo href indicando la pagina web a la que queremos crear el vinculo. Dentro de la etiqueta tendremos que poner el nombre del hipervinculo.
Ejemplo: 

			<a href="http://www.google.com">Enlace a Google</a>
Esto daria como resultado: Enlace a Google

Para crear hipervinculos internos (enlaces a otras partes del mismo documento) es parecido, aparte de el enlace necesitamos de otro componente, el "ancla", que será el lugar a donde vayamos al pulsar el enlace. Darse cuenta de que en el nombre del enlace es necesario poner una almohadilla.
Ejemplo:

Ancla:

<a name="Inicio"></a>
Enlace:
<a href="#Inicio">Ir al inicio del documento</a>

Para ver como funcionan los hipervinculos internos el índice de este documento se han creado mediante este tipo de enlaces, al hacer click en alguno este nos lleva a la parte del manual que hemos seleccionado mediante el ancla.

Imagenes

Para insertar imagenes tenemos que usar la etiqueta img seguido del atributo src y en el valor escribir el lugar donde esta la imagen, teniendo en cuenta que depende de donde estemos, tambien podemos modificar la imagen introduciendo los siguentes atributos y su correspondiente valor: width (anchura), height (altura) y alt (texto descriptivo)
Ejemplo: 

			<img src="ruta_de_la_imagen.jpg">
Algunos formatos de imagen soportados son: png, gif, jpg o svg.

Datos tabulados: Tablas

Para crear una tabla lo primero es utlizar la etiqueta table y dentro de esta etiqueta crearemos las filas (tr) y columnas (td).

Ejemplo de una tabla de 2x3 (2 filas y 3 columnas) con el atributo border=1 en la etiqueta table para resaltar los bordes: 

			<table border=1>
				<tr>
						<td> Fila 1, Columna 1
						<td> Fila 1, Columna 2
						<td> Fila 1, Columna 3
					</tr>
					<tr>
						<td> Fila 2, Columna 1
						<td> Fila 2, Columna 2	
						<td> Fila 2, Columna 3
					</tr>
			
			
Resultado:

			

				

					 Fila 1, Columna 1
					 Fila 1, Columna 2
					 Fila 1, Columna 3
				

				

					 Fila 2, Columna 1
					 Fila 2, Columna 2	
					 Fila 2, Columna 3
